Differential Equations

An equation involving a function and one or more of its derivatives is called a differential equation. Differential equations are also defined as equations that contain derivatives of one or more dependent variables with respect to one or more independent variables. If a function has only one independent variable, then it is an ordinary differential equation.

Examples of Differential Equations:

1. \( \frac{d^2 y}{dx^2} - \frac{dy}{dx} - 4x = 0 \)
2. \( \left( \frac{d^2 y}{dx^2} \right) - 6 \frac{dy}{dx} + 8y = 0 \)
3. \( \left[ 2 + \left( \frac{dy}{dx} \right) \right] = \left( \frac{d^2 y}{dx^2} \right)^{\frac{2}{3}} \)

Note: A differential equation involving a function of several variables and its partial derivatives is called a partial differential equation.

Degree May Not Be Defined in Ways as Mentioned Below:

  • \( y' + 6y^2 + y = 0 \) at (1,1)
  • \( (y')^2 + y^2 - 1 = 0 \) at (1,2)
  • \( y'' + \left(2y_1\right)^6 + \sin(y) = 0 \) at (2,1)
  • \( y'' + \sin(y') + y = 0 \) at (2, ND)

Formation of Differential Equation

\( f(x,y,\alpha, \alpha^2, \ldots, \alpha^n) = 0 \)....(1)
Where, \( \alpha_1, \alpha_2, \ldots, \alpha_n \) are n different parameters. If we differentiate n times, we get the differential equation of the given equation (1).

Solution by Inspection

If the differential equation is of the form
\( f(f_1(x,y)) d(f_1(x,y)) + \phi(f_2(x,y)) d(f_2(x,y)) + \ldots = 0 \),
then each term can be separately integrated.

The solution to a differential equation can be found using the inspection method. It is accompanied by memorizing the following results:

  1. \(d(x+y) = dx + dy\)
  2. \(d(xy) = x \, dy + y \, dx\)
  3. \(d\left( \frac{x}{y} \right) = \frac{y \, dx - x \, dy}{y^2}\)
  4. \(d\left( \frac{y}{x} \right) = \frac{x \, dy - y \, dx}{x^2}\)
  5. \(d\left( \frac{x^2}{y} \right) = \frac{2x \, y \, dx - x^2 \, dy}{y^2}\)
  6. \(d\left( \frac{y^2}{x} \right) = \frac{2x \, y \, dy - y^2 \, dx}{x^2}\)
  7. \(d\left( \frac{x^2}{y^2} \right) = \frac{2x \, y^2 \, dx - 2x^2 \, y \, dy}{y^4}\)
  8. \(d\left( \frac{y^2}{x^2} \right) = \frac{2x^2 \, y \, dy - 2x \, y^2 \, dx}{x^4}\)
  9. \(d\left( \tan^{-1} \frac{x}{y} \right) = \frac{y \, dx - x \, dy}{x^2 + y^2}\)
  10. \(d\left( \tan^{-1} \frac{y}{x} \right) = \frac{x \, dy - y \, dx}{x^2 + y^2}\)
  11. \(d(\ln[xy]) = \frac{dx}{x} + \frac{dy}{y}\)
  12. \(d\left( \frac{x}{y} \right) = \frac{y \, dx - x \, dy}{y^2}\)
  13. \(d\left( \frac{1}{2} \ln(x^2 + y^2) \right) = \frac{x \, dx + y \, dy}{x^2 + y^2}\)
  14. \(d\left( \ln \frac{y}{x} \right) = \frac{x \, dy - y \, dx}{xy}\)
  15. \(d\left( \frac{-1}{xy} \right) = \frac{x \, dy + y \, dx}{x^2 y^2}\)
  16. \(d\left( \frac{e^x}{y} \right) = \frac{y \, e^x \, dx - e^x \, dy}{y^2}\)
  17. \(d\left( \frac{e^y}{x} \right) = \frac{x \, e^y \, dy - e^y \, dx}{x^2}\)
  18. \(d(x^m y^n) = x^{m-1} y^{n-1} (m y \, dx + n x \, dy)\)
  19. \(d(\sqrt{x^2 + y^2}) = \frac{x \, dx + y \, dy}{\sqrt{x^2 + y^2}}\)
  20. \(d\left( \frac{1}{2} \log \frac{x+y}{x-y} \right) = \frac{y \, dx - x \, dy}{x^2 - y^2}\)
